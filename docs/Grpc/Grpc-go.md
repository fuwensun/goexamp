## grpc Package

这个包是 grpc 的 go 实现.

```
# 国际标准安装
go get -u google.golang.org/grpc

# 国内从 github 镜像安装
git clone https://github.com/grpc/grpc-go.git $GOPATH/src/google.golang.org/grpc

# 查看
ls $GOPATH/src/google.golang.org/grpc
```
